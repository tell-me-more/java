06 · Wiederholen mit for

Eine for-Schleife ist gut, wenn du weißt, wie oft etwas laufen soll. Ein Zähler startet, läuft bis zu einer Grenze und erhöht sich Schritt für Schritt.

Kurz erklärt

Die for-Schleife wiederholt Code mit einem Zähler.

Schritt-für-Schritt
  1. Startwert festlegen.
  2. Bedingung festlegen.
  3. Zähler erhöhen.

Interaktives Beispiel

for (int i = 1; i <= 5; i++) {
  System.out.println(i);
}
(Noch keine Ausgabe)

Interaktive Übungen

Aufgabe 1

Schreibe eine for-Schleife von 1 bis 3.

Noch nicht geprüft.

Aufgabe 2

Schreibe eine for-Schleife von 2 bis 6.

Noch nicht geprüft.

Aufgabe 3

Gib mit for nur gerade Zahlen von 2 bis 10 aus.

Noch nicht geprüft.
