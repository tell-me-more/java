02 · Variablen

Variablen speichern Werte. Für Klasse 7 nutzen wir int, double und String.

Kurz erklärt

Variablen speichern Werte. Für Klasse 7 nutzen wir int, double und String.

Schritt-für-Schritt
  1. int ist für ganze Zahlen.
  2. String ist für Text.
  3. Variable links, Wert rechts.

Interaktives Beispiel

int punkte = 7;
System.out.println(punkte);
(Noch keine Ausgabe)

Interaktive Übungen

Aufgabe 1

Lege eine int-Variable alter an.

Noch nicht geprüft.

Aufgabe 2

Lege eine String-Variable name an.

Noch nicht geprüft.

Aufgabe 3

Lege int und String an und gib eine davon aus.

Noch nicht geprüft.
