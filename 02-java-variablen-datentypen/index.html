02 · Variablen und einfache Datentypen

Eine Variable ist wie eine beschriftete Box. In Klasse 7 reichen drei Typen: int für ganze Zahlen, double für Kommazahlen und String für Text.

Kurz erklärt

Eine Variable ist wie eine beschriftete Box. In Klasse 7 reichen drei Typen: int für ganze Zahlen, double für Kommazahlen und String für Text.

Ein Beispiel

int alter = 12;
double groesse = 1.58;
String name = "Lina";

Übungen

  1. Lege eine int-Variable punkte mit Wert 5 an.
  2. Lege eine String-Variable stadt mit deinem Ort an.
  3. Lege alle drei Typen korrekt in einem kurzen Programm an.
