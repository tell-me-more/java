04 · Entscheidungen mit if und else

Mit if prüfst du eine Bedingung. Ist sie wahr, läuft ein Block. Mit else legst du fest, was sonst passieren soll.

Kurz erklärt

if prüft eine Bedingung. else läuft, wenn die Bedingung falsch ist.

Schritt-für-Schritt
  1. Bedingung in runden Klammern.
  2. Codeblöcke in geschweiften Klammern.
  3. else ist der zweite Weg.

Interaktives Beispiel

int temp = 17;
if (temp < 20) {
  System.out.println("Jacke");
} else {
  System.out.println("T-Shirt");
}
