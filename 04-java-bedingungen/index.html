04 · Entscheidungen mit if und else

Mit if prüfst du eine Bedingung. Ist sie wahr, läuft ein Block. Mit else legst du fest, was sonst passieren soll.

Kurz erklärt

Mit if prüfst du eine Bedingung. Ist sie wahr, läuft ein Block. Mit else legst du fest, was sonst passieren soll.

Ein Beispiel

int temperatur = 17;
if (temperatur < 20) {
  System.out.println("Jacke anziehen");
} else {
  System.out.println("T-Shirt reicht");
}

Übungen

  1. Schreibe eine if-Abfrage für punktzahl > 10.
  2. Ergänze ein else mit einer zweiten Ausgabe.
  3. Nutze eine Variable note und gib zwei passende Texte aus.
