07 · Wiederholen mit while

Eine while-Schleife läuft, solange eine Bedingung wahr ist. Du musst im Block die Variable ändern, sonst endet die Schleife nicht.

Kurz erklärt

while läuft so lange, wie die Bedingung wahr ist.

Schritt-für-Schritt
  1. Startwert setzen.
  2. while-Bedingung schreiben.
  3. Im Block die Variable ändern.

Interaktives Beispiel

int n = 1;
while (n <= 4) {
  System.out.println(n);
  n++;
}
(Noch keine Ausgabe)

Interaktive Übungen

Aufgabe 1

Starte bei 1 und zähle bis 3.

Noch nicht geprüft.

Aufgabe 2

Starte bei 5 und zähle bis 8.

Noch nicht geprüft.

Aufgabe 3

Starte bei 10 und zähle bis 0 herunter.

Noch nicht geprüft.
