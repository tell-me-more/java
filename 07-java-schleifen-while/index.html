07 · Wiederholen mit while

Eine while-Schleife läuft, solange eine Bedingung wahr ist. Du musst im Block die Variable ändern, sonst endet die Schleife nicht.

Kurz erklärt

Eine while-Schleife läuft, solange eine Bedingung wahr ist. Du musst im Block die Variable ändern, sonst endet die Schleife nicht.

Ein Beispiel

int n = 1;
while (n <= 4) {
  System.out.println(n);
  n++;
}

Übungen

  1. Lass n von 1 bis 3 laufen.
  2. Starte bei 5 und zähle bis 8 hoch.
  3. Lass eine Variable bei 10 starten und bis 0 herunterzählen.
