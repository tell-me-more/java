05 · Vergleiche und logische Verknüpfungen

Vergleiche liefern wahr oder falsch, zum Beispiel ==, > oder <=. Mit && verbindest du zwei Bedingungen, die beide wahr sein müssen.

Kurz erklärt

Vergleiche liefern wahr oder falsch, zum Beispiel ==, > oder <=. Mit && verbindest du zwei Bedingungen, die beide wahr sein müssen.

Ein Beispiel

int alter = 13;
boolean darfMit = alter >= 12 && alter <= 15;
System.out.println(darfMit);

Übungen

  1. Prüfe, ob x genau 5 ist.
  2. Prüfe, ob eine Zahl größer 0 und kleiner 10 ist.
  3. Bilde eine Variable erlaubt mit zwei Bedingungen und gib sie aus.
