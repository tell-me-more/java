05 · Vergleiche

Vergleiche liefern wahr oder falsch. Mit && verknüpfst du zwei Bedingungen.

Kurz erklärt

Vergleiche liefern wahr oder falsch. Mit && verknüpfst du zwei Bedingungen.

Schritt-für-Schritt
  1. == bedeutet gleich.
  2. >= und <= vergleichen Grenzen.
  3. && bedeutet: beide Bedingungen müssen stimmen.

Interaktives Beispiel

int alter = 13;
boolean erlaubt = alter >= 12 && alter <= 15;
System.out.println(erlaubt);
(Noch keine Ausgabe)

Interaktive Übungen

Aufgabe 1

Prüfe, ob x genau 5 ist.

Noch nicht geprüft.

Aufgabe 2

Prüfe, ob n größer 0 und kleiner 10 ist.

Noch nicht geprüft.

Aufgabe 3

Lege boolean ok mit zwei Bedingungen an und gib ihn aus.

Noch nicht geprüft.
