08 · Häufige Fehler vermeiden

Viele Fehler sind kleine Schreibfehler. Prüfe immer Klammern, Semikolon und Groß-/Kleinschreibung. Lies den Code Zeile für Zeile.

Kurz erklärt

Typische Fehler sind fehlendes Semikolon, falsche Klammern oder falsche Schreibweise.

Schritt-für-Schritt
  1. Jede Befehlszeile endet mit ;
  2. Klammern immer paarweise setzen.
  3. Groß- und Kleinschreibung beachten.

Interaktives Beispiel

int zahl = 7;
if (zahl > 5) {
  System.out.println("ok");
}
(Noch keine Ausgabe)

Interaktive Übungen

Aufgabe 1

Schreibe eine korrekte Ausgabezeile mit Semikolon.

Noch nicht geprüft.

Aufgabe 2

Schreibe eine korrekte if-Bedingung mit Klammern.

Noch nicht geprüft.

Aufgabe 3

Schreibe drei korrekte Zeilen: int, if, Ausgabe.

Noch nicht geprüft.
