08 · Häufige Fehler vermeiden

Viele Fehler sind kleine Schreibfehler. Prüfe immer Klammern, Semikolon und Groß-/Kleinschreibung. Lies den Code Zeile für Zeile.

Kurz erklärt

Viele Fehler sind kleine Schreibfehler. Prüfe immer Klammern, Semikolon und Groß-/Kleinschreibung. Lies den Code Zeile für Zeile.

Ein Beispiel

int zahl = 7;
if (zahl > 5) {
  System.out.println("ok");
}

Übungen

  1. Finde in einem eigenen Beispiel ein fehlendes Semikolon.
  2. Schreibe eine if-Zeile einmal falsch und dann korrekt.
  3. Baue absichtlich drei typische Fehler ein und verbessere sie danach.
