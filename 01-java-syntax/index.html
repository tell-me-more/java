01 · Java-Syntax entdecken

In diesem Modul lernst du die Grundregeln der Java-Syntax Schritt für Schritt kennen. Lies zuerst die Merksätze und bearbeite danach die Erkundungsaufgaben.

Grundlagen

Erkundungsaufgaben

Fortschritt: 0 / 4 erledigt

Aufgabe 1: Zuordnen

Ordne jedem Begriff den passenden Java-Code zu.

Aufgabe 2: Verschieben

Bringe die Codezeilen in eine sinnvolle Reihenfolge. Nutze ↑ und ↓.

    Aufgabe 3: Fehlersuche

    Markiere die Zeilen mit Syntaxfehlern. Es gibt genau 3 fehlerhafte Zeilen.

    Aufgabe 4: Selbst Code schreiben

    Schreibe ein Mini-Programm, das eine Klasse Main enthält und in der main-Methode den Text "Java ist cool" ausgibt.
