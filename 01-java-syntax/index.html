01 · Start und erste Ausgabe

Kurz erklärt

Java-Programme laufen in einem festen Rahmen. Du musst den Rahmen nur sauber abschreiben. Wichtig ist heute nur: Eine Ausgabezeile endet mit Semikolon.

Ein Beispiel

public class Main {
  public static void main(String[] args) {
    System.out.println("Hallo");
  }
}

Übungen

  1. Schreibe ein Programm, das "Hi" ausgibt.
  2. Ändere die Ausgabe auf deinen Namen.
  3. Erweitere auf zwei Ausgaben untereinander.
