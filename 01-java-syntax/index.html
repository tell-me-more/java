01 · Start und erste Ausgabe

Java-Programme laufen in einem festen Rahmen. Du musst den Rahmen nur sauber abschreiben. Wichtig ist heute nur: Eine Ausgabezeile endet mit Semikolon.

Kurz erklärt

Ein Befehl kann direkt etwas in der Konsole ausgeben. Jede Zeile endet mit Semikolon.

Schritt-für-Schritt
  1. Schreibe genau System.out.println.
  2. Setze den Text in Anführungszeichen.
  3. Beende die Zeile mit ;

Interaktives Beispiel

System.out.println("Hallo");
(Noch keine Ausgabe)

Interaktive Übungen

Aufgabe 1

Schreibe eine Zeile, die "Hi" ausgibt.

Noch nicht geprüft.

Aufgabe 2

Gib deinen Namen aus.

Noch nicht geprüft.

Aufgabe 3

Gib zwei Wörter in einer Zeile aus.

Noch nicht geprüft.
